1) Explain the difference between int and float data types in C. When would you use one over the other?

Answer Responded

'int' is a data type in C used for storing integer values, which are whole numbers. It does not store decimal points. 'float', on the other hand, is a data type used for storing floating-point numbers, which include decimal points.

2)Describe the process of declaring and defining functions in C. How do you call a function, and what is its purpose?

Answer Responded

In C, functions are declared using a function prototype, which specifies the function's name, return type, and parameter list. Functions are defined with a function body that contains the actual code to be executed. To call a function, you use its name followed by parentheses, passing any required arguments. The purpose of functions in C is to encapsulate a specific piece of code for reusability, making the code more modular and easier to manage.
